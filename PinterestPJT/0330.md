## 8강 - 장고 Template extends include 구문, render

HTML을 사용한 장고 템플릿

- HTML
  - Hyper Text : 문서 간 이동이 가능한 텍스트
  - Markup Language



- django Template
  - extends
    - HTML을 가져온다.
    - 미리 만들어 놓은 HTML 파일을 바탕으로 안에 있는 블럭을 채워나간다.
    - 바탕을 깔아주는 느낌
  - include
    - HTML을 가져온다.
    - 조그만한 조각을 가져와서 탬플릿 안에다가 집어넣는다.
    - 더미를 가져와서 붙여넣는 느낌
